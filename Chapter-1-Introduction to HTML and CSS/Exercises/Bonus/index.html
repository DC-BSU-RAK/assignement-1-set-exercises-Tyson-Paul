HTML & CSS Quick Reference Guide

HTML Tags

<h1> to <h6> - Headings

Used to define headings, from most important (<h1>) to least important (<h6>).

<h1>Main Heading</h1>

<p> - Paragraph

Defines a paragraph of text.

<p>This is a paragraph.</p>

<a> - Anchor

Creates a hyperlink to another page or location.

<a href="https://example.com">Visit Example</a>

<img> - Image

Embeds an image into the page.

<img src="image.jpg" alt="Description" />

<ul>, <ol>, <li> - Lists

Used for unordered and ordered lists.

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

<div> - Division

A container used to group content.

<div class="container">
  <p>Inside a div</p>
</div>

<span> - Inline Container

Used to style small portions of text inline.

<p>This is <span style="color:red">red</span> text.</p>

<form> - Form

Creates an HTML form for user input.

<form action="submit.php" method="post">
  <input type="text" name="name" />
</form>

<input> - Input Field

Used inside forms to receive user input.

<input type="text" placeholder="Enter your name" />

<button> - Button

Clickable button for submitting forms or triggering JavaScript.

<button>Click Me</button>

CSS Properties

color

Sets the color of the text.

p { color: darkblue; }

background-color

Sets the background color of an element.

div { background-color: #f0f0f0; }

font-size

Sets the size of the text.

h1 { font-size: 2em; }

margin

Sets the space outside the element.

div { margin: 20px; }

padding

Sets the space inside the element.

div { padding: 15px; }

border

Sets the border around the element.

div { border: 1px solid #ccc; }

display

Controls the display type of the element.

span { display: inline-block; }

text-align

Aligns text horizontally.

h1 { text-align: center; }
